Containers

  • header: cabeçalho da página
  • nav: navegação da página
  • section: seções da página
  • footer: rodapé
  • aside: conteúdo extra que fica ao lado
  • article: conteúdos independetes da página
  • div: container genérico

Atributos

  • id: identificador único
  • href: indica a localização de um recurso
  • src: indica a localização de uma mídia
  • alt: texto alternativo
  • class: especificar classe css
  • style: permite adicionar css
  • title: tool tip (o negócio que aparece lido pelo cursor do mouse)
  • target: indica onde abrir um link
  • rel: indica a relação entre o documento atual e o recurso linkado
